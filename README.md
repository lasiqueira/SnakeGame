# SnakeGame

Snake Game in C++ and SDL 3

<img width="788" alt="snake" src="https://github.com/user-attachments/assets/aadd054e-c043-4dd6-a1f2-42805761e558" />

## Build instructions
```
mkdir build
cd build
cmake ..
make
```

## Running
```
./SnakeGame
```

## Controls

UP - w or up arrow

DOWN - s or down arrow

LEFT - a or left arrow

RIGHT - d or right arrow
